# Android Tutorial Code
Codebase containing tutorial android tutorial code for 2nd year Mobile Application Development module.

## Important Usage Notes:
- **Do not use the main branch.** It contains outdated code.
- To find the code for the corresponding week, change the branch to the week you're looking for.

## Tooling, Hardware and Software

Operating System : <br/>Windows / MacOS / Linux

Memory: <br/>Minimum 8GB of RAM (to support both Emulator and IDE) or an external Android device

IDE: <br/>Android Studio (written using Android Studio Bumblebee)

## Reporting Bugs

If you manage to find a bug in the code. Open an issue for it via the Issues Tab by using the proper tag alongside it. Please include the bug and the steps you used to reproduce the bug.
